Introduction

Floyd's algorithm is a cycle detection algorithm. Let \(S\) be any finite set and \(f\) be a function from \(S\) to itself. For any \(i > 0\) let \(x_i = f(x_{i-1})\). Let \(\mu\) be the smallest index such that the value \(x_\mu\) reappears infinitely often within the sequence of values \(x_i\) and let \(\lambda\) be the smallest positive integer such that \(x_\mu = x_{\mu + \lambda}\). Floyd's algorithm is concerned with finding \(\lambda\) and \(\mu\).

For example, in figure 1, we have a function \(f\) which maps the set \(S = \{0,1,2,3,4\}\) to itself. If one starts from \(x_0 = 0\) and repeatedly applies \(f\) one sees the sequence of values \(0, 1, 2, 3, 4, 1, 2, 3, 4, \dots\). As such, the cycle in this value sequence is \(1, 2, 3, 4\). The value of \(\mu\) is \(1\) and the value \(\lambda\) is \(4\).

Complexity

This code only accesses the sequence by storing and copying pointers, function evaluations, and equality tests; therefore, it qualifies as a pointer algorithm. The algorithm uses O(λ + μ) operations of these types, and O(1) storage space.
