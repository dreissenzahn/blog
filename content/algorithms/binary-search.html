Introduction

Binary Search

Binary search is a search algorithm that finds the position of a target value within a sorted array. Binary search begins by comparing an element in the middle of the array with the target value. If the target value matches the element, its position in the array is returned. If the target value is less than the element, the search continues in the lower half of the array. If the target value is greater than the element, the search continues in the upper half of the array. By doing this, the algorithm eliminates the half in which the target value cannot lie in each iteration.

Procedure

Let \(A\) be an array of length \(n\) with values \(A_0, A_1, \dots, A_{n-1}\) sorted such that \(A_0 \le A_1 \le \dots \le A_{n-1}\). To search for a target value \(T\), we proceed as follows:

  1. Set \(L\) to \(0\) and \(R\) to \(n-1\).
  2. If \(L > R\) terminate unsuccessfully.
  3. Set \(m\) to \(\left \lfloor{\frac{L+R}{2}}\right \rfloor\).
  4. If \(A_m < T\) set \(L\) to \(m+1\) and go to step 2.
  5. If \(A_m > T\) set \(R\) to \(m-1\) and go to step 2.
  6. Now \(A_m = T\) so return \(m\).

Time Complexity

  • Best-case: \(O(1)\)
  • Average-case: \(O(\log n)\)
  • Worst-case: \(O(\log n)\)
  • Worst-case space: \(O(1)\)
