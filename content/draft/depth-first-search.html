Introduction

Traversal

Depth-first search starts from a source vertex and repeatedly explores an unvisited vertex adjacent to the current vertex. This proceeds until the algorithm cannot find such an unexplored vertex at which point it backtracks along previously visited vertices until it finds an unvisited vertex to explore. It then continues down this new path until it encounters another dead-end at which point it backtracks. This continues until the algorithm backtracks past the source vertex.

Depth-first search proceeds as follows:

  1. Initialize an empty stack.
  2. Mark the source vertex as visited and push it onto the stack.
  3. Pop a vertex off the stack and consider each of its adjacent vertices.
  4. If a vertex adjacent to the popped vertex has not been visited then mark it as visited and push it onto the stack.
  5. Repeat steps 3-4 until the stack is empty.

Complexity

Both breath-first search and depth-first search have worst-case performance of \(O(|V| + |E|)\) and worst-case space complexity of \(O(|V|)\).
